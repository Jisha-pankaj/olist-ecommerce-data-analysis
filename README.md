# Olist E-commerce Data Analysis Project

## Objective
To analyze a real-world e-commerce dataset and derive business insights.

## Dataset
Brazilian E-Commerce Public Dataset by Olist (Kaggle)

Dataset link:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

Note: The raw CSV files are not uploaded here due to file size. 
They can be downloaded from Kaggle using the link above.

## Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## Key Questions Answered
1. Monthly sales trend
2. Top 10 product categories by revenue
3. Cities with the most customers
4. Average delivery time
5. Percentage of late deliveries
